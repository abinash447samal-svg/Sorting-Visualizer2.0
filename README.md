# Sorting Visualizer 2.0

An interactive **Sorting Algorithm Visualizer** built using HTML, CSS, and JavaScript.

## Features

*  Visualizes multiple sorting algorithms
*  Sound effects while sorting
*  Adjustable speed and array size
*  Displays **C code implementation**
*  Clean and modern UI

## Algorithms Included

* Bubble Sort
* Insertion Sort
* Selection Sort
* Quick Sort
* Merge Sort

## How to Run

1. Download or clone the repo
2. Open `sorting_visualizer_with_code.html` in your browser

## Future Improvements

* Add more algorithms
* Add multiple language support (Java/Python)
* Highlight executing code lines
